# üíñ Interactive Valentine Proposal

A playful, interactive web experience designed to ask that special someone to be your Valentine. This project features a "fleeing" No button, a live countdown, and a celebratory confetti finish.

## ‚ú® Features

* **Dynamic Personalization:** Pass a name through the URL to customize the proposal.
* **The Runaway "No" Button:** A cheeky script that prevents the user from clicking "No" by moving the button away on hover or touch.
* **Live Countdown:** Tracks the remaining time until February 14th in real-time.
* **Celebration:** Clicking "Yes" triggers a double-burst of confetti.
* **Fully Responsive:** Works beautifully on both desktop and mobile devices.

## üöÄ How to Use

You can send a personalized link to anyone without changing the code! Just add `?name=NameGoesHere` to the end of your GitHub Pages URL.

**Examples:**
* `https://siddharthsbhadauria.github.io/valentines/?name=Piyu`
* `https://siddharthsbhadauria.github.io/valentines/?name=Alex`
* `https://siddharthsbhadauria.github.io/valentines/?name=Soulmate`

*If no name is provided, the page defaults to "My Love".*
